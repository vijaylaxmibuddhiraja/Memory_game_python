# Memory_game_python
Ett konsolbaserat minnesspel byggt med Python där spelare testar sitt minne genom att komma ihåg och återge sekvenser av bokstäver eller ord.

# Hur man spelar
1. Skriv in ditt namn för att börja.
2. Välj en svårighetsgrad.
3. Memorera sekvensen innan timern når noll.
4.  skriv in sekvensen i rätt ordning.
5. Tjäna poäng för rätt svar och förlora poäng för fel svar

# Svårighetsgrader
1. Beginner (A): 5 slumpmässiga bokstäver (5 sekunder att memorera)
2. Intermediate (B): 7 slumpmässiga bokstäver (7 sekunder att memorera)
3. Expert (C): 5 slumpmässiga ord (10 sekunder att memorera)

# Funktioner
1. Poängsystem (+10 vid vinst, -5 vid förlust).
2. Möjlighet att spela om nivåer eller återvända till huvudmenyn.
